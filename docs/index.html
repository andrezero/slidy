Usage

Install with NPM


npm install @andrezero/slidy

And import it into your project.


import Slidy from '@andrezero/slidy';

Load via CDN

Alternatively, you can load Slidy into a web page directly.


<!-- goes before </head> -->
<link rel="stylesheet" type="text/css" href="//cdn.../slidy.css" />

<!-- goes before </body> -->
<script type="text/javascript" src="https://cdn.../slidy.js" ></script>

Attach & destroy

Add several elements to one container.


<div class="demo" id="usage-1">
  <div class="item">A</div>
  <div class="item">B</div>
  <div class="item">C</div>
</div>

Attach an instance of Slidy to the container element.


const slidy = Slidy.create($('#container'));

Destroy a carousel instance, restoring original DOM and removing all event listeners.


slidy.destroy();
